## How to add yours

1. Fork the repo.
2. Add an entry below using this template:

```
### <Project name>
- **Author:** Your name + link
- **Hardware:** ESP32-S3 / Pi 4 / etc.
- **What it does:** one sentence.
- **Demo:** link to video, blog, or photo.
- **Code:** link to your fork or downstream repo, if open.
```

3. Open a PR. We will merge fast.

If your build is closed-source for commercial reasons, that is fine. We still want to know it exists.
